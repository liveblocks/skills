# Yjs best practices

This guide covers best practices and common pitfalls to avoid when working with
Yjs and Liveblocks. IMPORTANT: This does NOT apply to using Tiptap, BlockNote,
or Lexical with Liveblocks. These packages use their own Yjs abstractions, which
makes this guide irrelevant.

[Learn more](https://liveblocks.io/docs/guides/yjs-best-practices-and-tips.md).
